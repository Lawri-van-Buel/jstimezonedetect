Automatic Timezone Detection Using JavaScript

This script is open source and hosted as jsTimezoneDetect @ Bitbucket.

How do I test this script?

To test the detection script, simply change your system timezone and refresh this page. In Chrome and Opera you will have to open a new tab (and sometimes restart) for the new timezone to be detected.

What is the Olson Database?

The Olson database, also known as the tz database. Is a public domain collaborative compilation of information about the world's timezones.

It is widely used as basis for timezone information in operating systems and software. Most platforms and programming languages have libraries that support it.

More at Wikipedia!

I don't like the timezone this script gives me!

This script does not do geolocation. It simply picks the most populated timezone among several roughly identical ones.

For example, it is impossible to distinguish between different cities in Europe. This script will return Europe/Berlin regardless of where you live in the Central European timezone.
